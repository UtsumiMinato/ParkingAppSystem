## branch:
main-線上分支  
dev-開發分支  
剩餘為個人分支與功能開發分支  

### 建立分支:
如果想要寫點東西可以建立自己的分支或功能分支，先切換到dev。  
在 Terminal / CMD 內切換到專案根目錄下輸入:  
```
git checkout -b 分支名稱
```
把編輯完成的檔案加入追蹤:
```
git add .
```
新增更新內容:
```
git commit -m "說明修改內容"
```
上傳GitHub:
```
git push -u origin 剛才新增的分支名稱
```

### 分支命名:
feature: 功能  
ex: feature/add-enter-car-plate
